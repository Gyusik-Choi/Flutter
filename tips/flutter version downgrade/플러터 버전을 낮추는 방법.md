# 플러터 버전을 낮추는 방법

```bash
flutter downgrade
flutter --version
```

<br>

터미널에 flutter를 입력하면 관련된 명령어들을 확인할 수 있다.

flutter downgrade는 현재 들어와있는 flutter channel의 가장 최신에 사용했던 버전으로 바꾼다.
